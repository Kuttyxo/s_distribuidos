# Waze Traffic Scraper

Scraper para extraer datos de tráfico de Waze en la Región Metropolitana.

## Requisitos

- Docker 20+
- Docker Compose 1.29+

## Instalación

1. Clonar repositorio
2. Crear archivo `.env` basado en `.env.example`
3. Ejecutar:

```bash
docker-compose up -d --build
